# TruckIQ Fleet Monitor

Real-time fleet tracking system that ingests GPS data and visualizes trucks on a live map.
<img width="1306" height="983" alt="Screenshot 2026-04-21 011902" src="https://github.com/user-attachments/assets/0bce0484-18b0-49fa-ae2d-9672f0f4d718" />
## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB Atlas
- Socket.IO
- Leaflet (Maps)

## Features
- Live GPS tracking
- Real-time updates via WebSockets
- Interactive map visualization
- REST API backend

## Run Locally

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
